# Rank Server
- read database and show top ranks

## Used techniques
- Spring boot
- M2 database
- Thymeleaf
- React

## How to start
1. execute `build.bat` file (for window os)
1. run `src/main/java/com/aswinblue/RankServer/RankServerApplication.java`
